## 에이전트 지원 범위 정리
- 주로 사업체에서 사용하는 인허가 유형인 도시개발, 환경, 의료, 공장, 설비 등의 범위까지 고려하는 것은 MVP 개발 단계에서 어려움
- 하기 토지와 건축 등 개인이 주로 활용할만한 범위로만 축소

### 1. 토지 관련 인허가
| 인허가 | 개인 수요 |
| --- | --- |
| 개발행위허가 | 매우 높음 |
| 농지전용허가 | 매우 높음 |
| 산지전용허가 | 높음 |
| 도로점용허가 | 높음 |
| 하천점용허가 | 중간 |
| 국유지 사용허가 | 낮음 |
| 문화재 현상변경허가 | 낮음 |

### 2. 건축 인허가
| 인허가 | 개인 수요 |
| --- | --- |
| 건축허가 | 매우 높음 |
| 건축신고 | 매우 높음 |
| 증축신고 | 높음 |
| 대수선신고 | 높음 |
| 용도변경 | 높음 |
| 해체허가 | 중간 |
| 가설건축물 신고 | 높음 |
